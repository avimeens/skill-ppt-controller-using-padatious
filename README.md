## Ppt Controller Using Padatious
Control ppt presentation

## Description
Use your voice to open ppt presentation, navigate through the presentation (next, prev etc.) and final end it.

## Examples
 * ""open file abc.ppt""
 * ""open presentation mypresentation""
 * ""next slide""
 * ""next""
 * ""previous slide""
 * ""previous""
 * ""move on""
 * ""go back""
 * ""end presentation""
 * ""close file""
 * ""bye""
